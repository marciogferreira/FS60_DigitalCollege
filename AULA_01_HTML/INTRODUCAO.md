# 🌐 Introdução ao Desenvolvimento Web

No desenvolvimento web, existem três principais áreas de atuação:

* 🎨 **Frontend**
* ⚙️ **Backend**
* 🔄 **Fullstack**

Cada uma tem responsabilidades e tecnologias específicas.

---

# 🎨 Frontend

## 📌 O que é?

O **Frontend** é a parte visual da aplicação.
É tudo aquilo que o usuário **vê e interage** no navegador.

Exemplos:

* Botões
* Formulários
* Menus
* Layout
* Animações
* Responsividade

---

## 🛠 Principais Tecnologias

### 🔹 HTML

Estrutura da página.

### 🔹 CSS

Estilização (cores, tamanhos, layouts).

### 🔹 JavaScript

Interatividade e comportamento da página.

---

## 🚀 Frameworks e Bibliotecas Frontend

* React
* Vue.js
* Angular
* Next.js
* Bootstrap
* Tailwind CSS

---

## 👨‍💻 O que um Desenvolvedor Frontend faz?

* Implementa layouts
* Converte design (Figma) em código
* Integra APIs
* Garante responsividade
* Melhora performance da interface

---

# ⚙️ Backend

## 📌 O que é?

O **Backend** é a parte que roda no servidor.
Ele é responsável por:

* Processar regras de negócio
* Conectar com banco de dados
* Autenticação de usuários
* Segurança
* Enviar e receber dados da API

O usuário não vê o backend diretamente.

---

## 🛠 Principais Tecnologias Backend

### 🔹 Linguagens

* JavaScript (Node.js)
* Python
* Java
* C#
* PHP
* Go

---

### 🔹 Frameworks

* Express
* NestJS
* Django
* Spring Boot
* Laravel
* ASP.NET

---

### 🔹 Bancos de Dados

* MySQL
* PostgreSQL
* MongoDB
* SQL Server
* Firebase

---

## 👨‍💻 O que um Desenvolvedor Backend faz?

* Cria APIs
* Modela banco de dados
* Implementa autenticação
* Garante segurança
* Trabalha com servidores e deploy

---

# 🔄 Fullstack

## 📌 O que é?

O **Fullstack** é o profissional que trabalha tanto no Frontend quanto no Backend.

Ele consegue:

* Criar a interface
* Desenvolver a API
* Conectar banco de dados
* Fazer deploy da aplicação

---

## 🛠 Tecnologias Fullstack Populares

### 🔥 JavaScript Fullstack

* Frontend: React / Next.js
* Backend: Node.js / Express
* Banco: MongoDB ou PostgreSQL

### 🐍 Python Fullstack

* Frontend: React ou Templates
* Backend: Django / Flask
* Banco: PostgreSQL

### ☕ Java Fullstack

* Frontend: Angular ou React
* Backend: Spring Boot
* Banco: MySQL ou PostgreSQL

---

# 🧠 Comparação Rápida

| Área      | Foco Principal | Trabalha com Interface? | Trabalha com Servidor? |
| --------- | -------------- | ----------------------- | ---------------------- |
| Frontend  | Visual         | ✅ Sim                   | ❌ Não                  |
| Backend   | Regras e Dados | ❌ Não                   | ✅ Sim                  |
| Fullstack | Ambos          | ✅ Sim                   | ✅ Sim                  |

---

# 🏁 Qual escolher?

Depende do seu perfil:

* 🎨 Gosta de design e interação? → Frontend
* ⚙️ Gosta de lógica e sistemas? → Backend
* 🔄 Quer dominar tudo? → Fullstack
