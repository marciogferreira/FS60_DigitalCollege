O que é JavaScript?

JavaScript é uma linguagem de programação utilizada para tornar páginas web interativas e dinâmicas.

  • Executa no navegador
  • Manipula HTML e CSS
  • Responde a eventos do usuário
  • Pode ser usado também no servidor (Node.js)

O que são Variáveis?

Variáveis são espaços na memória utilizados para armazenar dados. 


let nome = "Maria";

A variável nome armazena o texto "Maria".

Declaração de Variáveis


var cidade = "Fortaleza";

let idade = 20;

const PI = 3.14159;
  • var → forma antiga
  • let → variável que pode mudar
  • const → valor constante

Diferença entre let e const


let contador = 1;
contador = 2;

const taxa = 10;
// taxa = 20; // erro

Use const sempre que o valor não precisar mudar.

Tipos de Dados

  • String
  • Number
  • Boolean
  • Null
  • Undefined
  • Object
  • Array

String

Representa textos.


let nome = "João";
let curso = 'JavaScript';

Podem ser usadas aspas simples ou duplas.

Number

Representa números inteiros ou decimais.


let idade = 25;

let altura = 1.75;

Boolean

Representa verdadeiro ou falso.


let aprovado = true;

let logado = false;

Null

Representa ausência intencional de valor.


let usuario = null;

Undefined

Valor padrão de uma variável não inicializada.


let endereco;

console.log(endereco);

Object

Estrutura para armazenar múltiplos dados.


let aluno = {
    nome: "Ana",
    idade: 18
};

Array

Lista de elementos.


let frutas = [
    "Maçã",
    "Banana",
    "Laranja"
];

Descobrindo o Tipo


let nome = "Carlos";

console.log(typeof nome);
