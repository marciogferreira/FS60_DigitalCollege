O que é Lógica de Programação?

É a técnica de organizar pensamentos e instruções para resolver problemas de forma estruturada.

Programar é ensinar o computador a executar uma sequência de passos.

Por que estudar lógica?

  • Resolver problemas de forma eficiente
  • Desenvolver raciocínio lógico
  • Facilitar o aprendizado de linguagens
  • Criar algoritmos corretos

O que é um Algoritmo?

Um algoritmo é uma sequência finita de passos para resolver um problema.

Exemplo: Fazer Café

  1. Colocar água na cafeteira
  2. Adicionar café
  3. Ligar a cafeteira
  4. Aguardar o preparo
  5. Servir o café

Estruturas de Repetição

PARA i DE 1 ATÉ 5 FAÇA
  Exibir(i)
FIMPARA

Utilizadas para repetir instruções várias vezes.
