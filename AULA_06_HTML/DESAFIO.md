
# 💻 Desafio: Criação de Site Completo com HTML e CSS

## 🎯 Objetivo

Desenvolver um site completo utilizando **apenas HTML** (obrigatório) e **CSS** (opcional, mas recomendado), com múltiplas páginas interligadas por um menu de navegação.

---

## 📁 Estrutura do Projeto

O site deve conter **4 páginas obrigatórias**:

* `index.html` (Página inicial)
* `quem-somos.html`
* `produtos.html`
* `contato.html`

---

## 📌 Requisitos Gerais

### 🧱 Estrutura

* Utilizar HTML sem frameworks (sem Bootstrap, sem JavaScript)
* CSS pode ser usado para estilização
* Criar um **menu de navegação** presente em todas as páginas
* As páginas devem estar interligadas corretamente

---

## 📄 Conteúdo de Cada Página

### 🏠 `index.html` (Página Inicial)

* Título do site
* Texto de apresentação
* Pelo menos **1 imagem**
* Destaque de conteúdo (ex: banner ou seção principal)

---

### 👥 `quem-somos.html`

* Texto descrevendo a empresa ou projeto
* Missão, visão e valores (ou equivalente)
* Pelo menos **1 imagem ilustrativa**

---

### 🛍️ `produtos.html`

* Lista de produtos ou serviços
* Criar uma **tabela HTML** com pelo menos:

  * Nome do produto
  * Descrição
  * Preço
* Pode incluir imagens dos produtos

---

### 📞 `contato.html`

* Criar um **formulário HTML** contendo:

  * Nome
  * Email
  * Assunto
  * Mensagem
* Botão de envio
* Informações adicionais de contato (ex: telefone, endereço)

---

## 🎨 Estilização (Opcional, mas recomendado)

* Utilizar CSS para:

  * Cores
  * Fontes
  * Espaçamento
  * Layout (menu, seções, etc.)

---

## 🔗 Navegação

Todas as páginas devem conter um **menu de navegação** com links para:

* Home
* Quem Somos
* Produtos
* Contato

---

## 🚀 Extras (Diferencial)

Se quiser ir além, você pode incluir:

* Layout responsivo (mesmo sem media queries complexas)
* Melhor organização visual com CSS
* Uso de listas (`<ul>`, `<ol>`)
* Ícones ou melhorias visuais

---

## ✅ Critérios de Avaliação

* Organização do código
* Uso correto de tags HTML
* Navegação funcional entre páginas
* Presença de todos os elementos solicitados (imagens, tabela, formulário)
* Clareza e criatividade no conteúdo

---

## 📦 Entrega

* Pasta contendo todos os arquivos HTML
* Arquivo CSS (se utilizado)
* Imagens utilizadas no projeto

---

Boa sorte e bom código! 🚀
